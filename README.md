# meshController
meshController is a Maya C++ plugin that provides high-performance surface pinning and mesh deformation tooling for production facial and character rigging

## Documentation
[https://www.cgdzg.com/docs/meshController-docs.html](https://www.cgdzg.com/docs/meshController-docs.html)

## Installation

1. Copy `meshControlManagerUI.py` into your Maya scripts folder:
   ```
   C:\Users\<username>\Documents\maya\2024\scripts\
   ```

2. Copy `meshController.mll` into the Maya plug-ins folder:
   ```
   C:\Program Files\Autodesk\Maya2024\bin\plug-ins\
   ```
